# 开封文旅与华创证券合作方案 (Kaifeng Culture & Tourism x Huachuang Securities)

本项目是一个基于 React + Tailwind CSS 构建的交互式合作方案展示页面，旨在展示开封文旅与华创证券在文化金融、智慧旅游及品牌联动方面的深度合作。

## 项目特点

- **沉浸式视觉体验**：采用开封清明上河园实景背景，融合宋代美学与现代金融感。
- **响应式设计**：完美适配桌面端与移动端。
- **动态交互**：使用 Framer Motion 实现流畅的入场动画与交互效果。
- **模块化架构**：基于 React 组件化开发，易于维护与扩展。

## 快速开始

### 环境要求

- Node.js (建议 v18+)
- npm 或 yarn

### 安装依赖

```bash
npm install
```

### 本地开发

```bash
npm run dev
```

### 构建生产版本

```bash
npm run build
```

## 技术栈

- **前端框架**: React 18
- **样式工具**: Tailwind CSS
- **动画库**: Framer Motion
- **图标库**: Lucide React
- **构建工具**: Vite
